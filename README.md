# 星際大冒險 (Interstellar Adventure)

一款基於 HTML5 Canvas 的 2D 橫向捲軸網頁小遊戲。結合了飛行避障與密室穿梭元素。

## 遊戲特色
- **超慢速重力感**：極具放鬆感的星際漂浮飛行體驗。
- **3D 視覺風格**：具備打光效果的齒輪、水管與傳送門。
- **傳送機制**：穿梭於場景中的傳送水管，快速跳躍距離。
- **隨機生成**：動態生成的障礙物與關卡配置。

## 操作方式
- **上方向鍵 (Arrow Up) / 空白鍵 (Space)**：向上推進。
- **目標**：避開齒輪與管壁，抵達地圖末端的發光傳送門 (Portal) 即可獲勝。

## 如何執行
1. 下載此專案。
2. 使用瀏覽器開啟 `index.html`。
3. 或者使用本地伺服器執行（例如 `python -m http.server 8000`）。
